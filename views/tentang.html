Klasifikasi Kecanduan Game Online

Sistem berbasis metode Fuzzy Tsukamoto untuk mengidentifikasi tingkat kecanduan berdasarkan gejala dan perilaku pengguna.

Tentang Sistem

Sistem ini dirancang untuk membantu mengklasifikasikan tingkat kecanduan game online pada individu berdasarkan beberapa indikator perilaku.

Dengan menggunakan pendekatan logika fuzzy metode Tsukamoto, sistem dapat menghasilkan keputusan yang lebih fleksibel dan mendekati kondisi nyata.

Tujuan Sistem

  • ✓ Membantu mendeteksi tingkat kecanduan game online
  • ✓ Memberikan klasifikasi: Rendah, Sedang, Tinggi
  • ✓ Memberikan hasil analisis berdasarkan perhitungan fuzzy
  • ✓ Mendukung pengambilan keputusan preventif

Metode Fuzzy Tsukamoto

Metode Fuzzy Tsukamoto merupakan salah satu metode dalam logika fuzzy yang menggunakan aturan berbentuk IF-THEN dengan fungsi keanggotaan monoton. Setiap aturan menghasilkan nilai crisp yang kemudian dihitung menggunakan rata-rata terbobot.

1. Fuzzifikasi

Mengubah nilai input menjadi derajat keanggotaan fuzzy.

2. Pembentukan Aturan

Menerapkan aturan IF-THEN berdasarkan gejala.

3. Inferensi

Menghitung nilai alpha-predikat dari setiap aturan.

4. Defuzzifikasi

Menghasilkan output crisp berupa tingkat kecanduan.
